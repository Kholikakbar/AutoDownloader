# Auto Downloader (Backend)

Backend service for Auto Downloader using Node.js, yt-dlp, and ffmpeg.
Designed to run on Hugging Face Spaces (Docker).

## Features
- Universal Video Downloader (YouTube, TikTok, Instagram, etc)
- Format Selection (Video/Audio)
- Quality Selection
- Supabase Logging Integration

## Deployment
This repo is configured for automated deployment on Hugging Face Spaces.
